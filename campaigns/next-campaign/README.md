# 新团资料区

## 状态

- 状态：筹备中。
- 玩家数：3。
- 起始等级：20 级。
- 角色、世界观、规则来源、属性生成与起始装备：待共同确认。

## 角色卡

三位玩家的独立角色卡位于 [characters/](characters/README.md)。每人只维护自己的文件；共享剧情、地图、任务和 DM 裁定将在开团后放在本目录中。

玩家 1 的创建草案为 [瑟伦·维尔 / Serren Vale](characters/serren-vale.md)：20 级人类预言学派法师。该卡是开团构筑，不是已发生的战役事实；起始资源、物品价格、基地位置和所有团桌裁定仍以 DM 确认为准。

## 基地预案

固定基地的分层警戒、施法区、睡眠区与撤离节点见 [基地建设说明](base-plan.md)。其中的法术与物品仅表示可用预案，不假定它们已经布设或必然对传奇敌人有效。

## 已确认团桌规则

新团的超 20 级成长、低价耗材、神格碎片与 CR 推进规则记录在 [团桌规则](rulings.md)。角色卡中的“后 20 级进阶”均以该文件为准。

## 计划目录

- `characters/`：三位玩家的 20 级角色卡与通用模板。
- `planning/`：构筑原稿、审阅结论与待 DM 确认的资源假设。
- `base-plan.md`：固定基地的共享安全预案与结构图。
- `rulings.md`：已确认的本团超 20 级成长与资源规则。
- `campaign.md`：开团后记录小队、任务、NPC 与冒险日志。
- `maps.md`：开团后记录玩家已探明的地图。
- `rulings.md`：仅记录本团专用或新确认的团桌裁定。

未创建的文件在本团正式开始时再加入，避免用猜测占据事实记录。
