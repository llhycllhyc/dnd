# 瑟伦·维尔构筑审阅

> 此文件解释 [瑟伦·维尔 / Serren Vale](../characters/serren-vale.md) 相对于原始构筑的调整。已确认的超 20 级规则见 [团桌规则](../rulings.md)；物品配额、价格、可抄录法术范围、基地位置与战役前准备时间以外的事项仍待 DM 确认。

## 已修正的原稿问题

- 物品购买实际合计是 **17,200 gp**，而不是章节标题中的 16,800 gp。
- 原始法术耗材合计是 **12,140 gp**，而不是章节标题中的 10,640 gp。按新团规则，守卫刻文、秘法锁、魔法嘴、寻获魔宠、真知术与防护善恶的单次消耗均低于 500 gp，因此不再占用金币预算；调整后的付费材料为 **9,615 gp**。
- 《清晰思维之书 / Tome of Clear Thought》现在允许先使用一次再出售。瑟伦保持人类，智力由 21 提高到 **23**，同时仍可获得 40,000 gp 资源池。
- 基地计划会用到的**莫登凯宁的忠实猎犬 / Mordenkainen's Faithful Hound**与**徽记术 / Symbol**原先没有写入法术书。现以同环法术替换，保持抄录总环数与价格不变：
  - **黑触手术 / Evard's Black Tentacles**替换为**忠实猎犬**；
  - **变大缩小术 / Enlarge/Reduce**与**异界束缚 / Planar Binding**替换为**徽记术**；
  - **加速术 / Haste**替换为**移除诅咒 / Remove Curse**，为未知敌人的诅咒、附身与条件风险预留解法。
  - **人类定身术 / Hold Person**与**暗示术 / Suggestion**替换为**秘法锁 / Arcane Lock**与**魔法嘴 / Magic Mouth**，使基地预案实际拥有这些法术，而不额外增加预算。

## 默认准备策略

20 级法师通常只能准备 25 个法术；“多抄录”不等于“每天都能准备”。因此默认清单优先放入三类法术：

1. 立即保命与脱离：护盾术、法师护甲、羽落术、迷踪步、飞行术、任意门、传送术。
2. 未知敌人的通用应对：防护善恶、解除魔法、移除诅咒、反制法术、法术无效结界、心灵屏障、预见术。
3. 传奇单体与战场控制：力墙术、力场囚笼、迷宫术、律令死亡、突触静电、祈愿术。

**预见术 / Foresight**已放入默认准备，取代高等级时更偏向临时变形用途的**变形术 / Polymorph**。这使“未知敌人”的每日开局更稳健：可以先把预见术给最可能遭受核心攻击的盟友或自己；若拥有拟像，拟像可负责预见术或心灵屏障，进一步腾出准备位。

**防护善恶**的 2024 版本仍会消耗一瓶价值至少 25 gp 的圣水，但在本团已确认为免费耗材。它仍需要专注，并只在敌人确属异怪、天界生物、元素、妖精、邪魔或不死生物时作为高价值防护使用，而不是每场战斗的常驻增益。

例外清单应在出发前更换，而不是试图覆盖所有敌人：

| 情报 | 加入 | 可先移出 |
|---|---|---|
| 敌方隐形、刺客或施法者 | **识破隐形 / See Invisibility**、**真实视觉 / True Seeing**、**解离术 / Disintegrate** | 蛛网术、催眠图纹、缓慢术 |
| 大量杂兵 | **流星爆 / Meteor Swarm**、**连锁闪电 / Chain Lightning**、**反重力 / Reverse Gravity** | 放逐术、力场囚笼、迷宫术 |
| 长途探索或无法返程 | **莫登凯宁豪宅术 / Mordenkainen's Magnificent Mansion**、**传送法阵 / Teleportation Circle** | 局部战斗法术 |
| 已知反魔法敌人 | **非魔法撤离、队友掩护、预先布置的警报** | 不把反魔法力场当作常规防护；它同样会压制己方法术与多数魔法物品 |

## 基地防护的规则边界

- **守卫结界 / Guards and Wards**、**防护结界 / Private Sanctum**、**警报术 / Alarm**、**秘法锁 / Arcane Lock**与**魔法嘴 / Magic Mouth**提供预警、阻滞和信息优势，但不是“敌人无法进入”的绝对保证。低价耗材免费不等于无视施法时间、法术位、固定地点或敌人的反制能力。
- **移位斗篷 / Cloak of Displacement**在受到伤害后会暂时失去攻击劣势效果；应把它视为第一击防线，而非无敌防御。
- **预置术 / Contingency**搭配需要专注的法术，不能作为“角色失去意识后仍维持法术”的可靠方案。睡眠区依靠警报、猎犬、守望武器和分隔空间来争取苏醒与撤离时间。若使用预置术搭配**任意门 / Dimension Door**，触发条件和预设目的地应先由 DM 确认。
- **力场囚笼 / Forcecage**需要价值 1,500 gp 的红宝石粉尘，每次使用都消耗；只为决定性目标保留一份材料。
- 92 个法术的法术书可能需要分为数卷。实际页数、抄录时间、备用书是否可在开局完成，应由 DM 明确。

## 后 20 级成长路线

每次胜利都会使下一名主要敌人的 CR +2，因此先拿立即生效的防御、紧急救援和移动能力，而不是追求需要长期投入才会启动的兼职子职。

| 成长等级 | 推荐兼职 | 收益 | 说明 |
|---:|---|---|---|
| 1 | 战士 1 / Fighter 1 | 中甲、盾牌、武术武器；**防御 / Defense**战斗风格；二次回气 / Second Wind | 取得精金半身甲与盾牌后，常态 AC 20；优先应对首个 CR +2 敌人。 |
| 2 | 牧师 1 / Cleric 1（奇术师 / Thaumaturge） | 指导术、临终救援、抗性术；4 个牧师准备法术 | 选择治疗真言、祝福术、庇护术、侦测善恶等不依赖高感知 DC 的法术。 |
| 3 | 游荡者 1 / Rogue 1 | 两项专精、盗贼黑话、额外语言 | 建议察觉与隐匿专精；提高基地反潜入与探索可靠性。 |
| 4 | 游荡者 2 / Rogue 2 | 狡诈动作 / Cunning Action | 奖励动作疾走、撤离或躲藏，让主动作始终可用于施法。 |
| 5 | 战士 2 / Fighter 2 | 动作如潮、战术心智 / Action Surge, Tactical Mind | 2024 动作如潮的额外动作**不能**用于魔法动作；价值在于施法后冲刺、闪避、利用物品或以二次回气给关键检定加 1d10。 |
| 6+ | 牧师 2 或按神格选择 | 引导神力，或转入与神格相关的兼职 | 若先获得神格碎片，则可优先采用下方的传奇恩惠路线。 |

### 神格碎片优先级

1. **次元旅行恩惠 / Boon of Dimensional Travel**：魅力 12 → 13，满足术士、邪术师或吟游诗人兼职门槛；每次魔法动作后的 30 尺传送也直接提高生存能力。同调上限变为 4。
2. **法术回忆恩惠 / Boon of Spell Recall**：智力 23 → 24，法术 DC 21、法术攻击 +13、调查 +19；1–4 环法术有机会不消耗法术位。
3. **真实视觉恩惠 / Boon of Truesight**：常驻真实视觉优先于继续堆叠奇数属性；同调上限变为 6。
4. **能量抗性恩惠 / Boon of Energy Resistance**：根据下一战已知伤害类型每日调整两种抗性。

首枚神格碎片后，若队伍确实需要更强的施法改造，再把后续兼职机会投向术士 2；否则优先完成牧师与游荡者的即时生存路线。术士的天生魔法只强化术士法术，不能直接提高法师法术 DC，因此不应在获得碎片前抢占战士 1 的位置。

## 开团前需要 DM 确认

1. 极珍稀物品是否可以按 40,000 gp 变现，以及魔法物品购买价是否采用原稿的固定表。
2. 可否在开局前以金币、时间和法术位抄录修正后列出的 39 个额外法术（130 个法术环阶）。
3. 基地是否已存在、可否预先施放长期法术、以及防护设施会否跨越开团后的第一场战斗保留。
4. **拟像术 / Simulacrum**是否允许开局已完成；若允许，拟像的装备、法术位和行动权按本团如何处理。
5. **预置术 / Contingency**的触发条件、与任意门的目的地选择方式，以及传奇刺客是否能在长休中绕过常规守卫。
