## 0. 战役前提与关键裁定  
## Campaign Assumptions and Key Rulings

- 使用**标准矩阵 / Standard Array**：15、14、13、12、10、8。
- 初始获得：
  - 一件**极珍稀物品 / Very Rare Item**；
  - 两件**珍稀物品 / Rare Items**。
- 极珍稀物品可以兑换 **40,000 gp**。
- 额外购买价格：
  - **普通 / Common**：100 gp；
  - **非普通 / Uncommon**：400 gp；
  - **珍稀 / Rare**：4,000 gp；
  - 不能额外购买极珍稀或更高物品。
- 可以用金币任意抄录法师法术，但必须遵守：
  - 法术在2024《玩家手册》中；
  - 角色已经拥有能够准备该法术的法术位环阶。
- 战斗节奏通常是：
  1. 从固定基地出发；
  2. 完成一场高强度战斗；
  3. 搜索神格碎片；
  4. 传送回基地；
  5. 完成长休。
- 神格碎片需要一次 **智力（调查）检定 / Intelligence (Investigation) Check**，总值达到30。
- 队友通常可以使用**协助 / Help**提供优势，牧师可以施放**指导术 / Guidance**增加1d4。
- 长休时可能遭遇传奇刺客，因此基地必须包含反潜入、预警、缓冲与复活措施。

# 1. 《清晰思维之书》的处理  
# Tome of Clear Thought Decision

## 1.1 主方案：人类，不获得书的智力加成

**推荐主方案：**

- 选择**人类 / Human**；
- 不阅读**《清晰思维之书》/ Tome of Clear Thought**；
- 直接将仍有魔力的书兑换为 **40,000 gp**；
- 最终不获得智力+2。

原因：

1. 阅读后，书会失去魔力，并在100年后恢复。
2. 若必须让书恢复魔力后才能按40,000 gp出售，就需要角色等待100年。
3. 一旦允许长寿角色通过等待100年获得一次智力+2并卖出恢复后的书，规则上同样容易推导出：
   - 每100年重新阅读一次；
   - 反复提高智力；
   - 最终把智力提高到30后再出售。
4. 这会形成远超正常开局资源的循环漏洞。
5. 人类本身提供：
   - **足智多谋 / Resourceful**：每次长休获得英雄激励；
   - **技能娴熟 / Skillful**：额外技能熟练；
   - **多才多艺 / Versatile**：额外一个起源专长。
6. 对“一战一长休”的赛制而言，人类每天恢复的英雄激励与额外的**幸运儿 / Lucky**非常强。

因此，除非DM明确允许并限制“百年读书方案”，主构筑不计算书的智力+2。

## 1.2 可选方案：高等精灵百年方案

只有DM明确同意以下全部条件时才采用：

- 可以在游戏开始前等待100年；
- 只允许角色阅读一次；
- 禁止同一角色每100年重复阅读；
- 100年后书恢复全部魔力；
- 恢复后的书仍可以按40,000 gp出售。

此时可以选择：

- **高等精灵 / High Elf**；
- 阅读《清晰思维之书》获得智力+2；
- 等待100年；
- 将恢复魔力的书出售；
- 最终智力从21提高到23。

高等精灵的额外收益：

- **黑暗视觉 / Darkvision**；
- **妖精血统 / Fey Ancestry**；
- **敏锐感官 / Keen Senses**；
- **出神 / Trance**：4小时完成长休且保持意识；
- **精灵血统 / Elven Lineage**：
  - 戏法：**魔法伎俩 / Prestidigitation**；
  - 3级：**侦测魔法 / Detect Magic**；
  - 5级：**迷踪步 / Misty Step**。

代价是失去人类的：

- 每次长休英雄激励；
- 一个额外技能；
- 一个额外起源专长，通常是**幸运儿 / Lucky**。

### 结论

- **常规、稳妥、不过度利用团规：人类 / Human。**
- **DM明确批准百年单次方案：高等精灵 / High Elf略强于法术DC与调查，但规则风险更高。**

以下正文以**人类主方案**为准。

# 2. 种族、背景、属性与技能  
# Species, Background, Ability Scores, and Skills

## 2.1 种族：人类 / Human

种族能力：

- **足智多谋 / Resourceful**  
  每次完成长休获得一次**英雄激励 / Heroic Inspiration**。

- **技能娴熟 / Skillful**  
  获得一个额外技能熟练，选择：
  - **调查 / Investigation**。

- **多才多艺 / Versatile**  
  获得一个额外起源专长，选择：
  - **幸运儿 / Lucky**。

## 2.2 背景：罪犯 / Criminal

主要收益：

- 起源专长：
  - **警觉 / Alert**。
- 属性提升：
  - 智力+2；
  - 敏捷+1。
- 背景技能：
  - **隐匿 / Stealth**；
  - **巧手 / Sleight of Hand**。
- 工具：
  - **盗贼工具 / Thieves’ Tools**。

## 2.3 标准矩阵分配

| 属性 / Ability | 初始 / Base | 背景后 / After Background | 20级自然值 / Natural at 20 | 装备后 / With Items |
|---|---:|---:|---:|---:|
| 力量 / Strength | 8 | 8 | 8 | 8 |
| 敏捷 / Dexterity | 13 | 14 | 14 | 14 |
| 体质 / Constitution | 10 | 10 | 11 | **19** |
| 智力 / Intelligence | 15 | 17 | **21** | 21 |
| 感知 / Wisdom | 14 | 14 | 14 | 14 |
| 魅力 / Charisma | 12 | 12 | 12 | 12 |

### 为什么体质只分配10？

因为免费珍稀物品选择：

- **健康护符 / Amulet of Health**：佩戴时体质固定为19。

因此不需要把标准矩阵中的14或13投入体质。

但不建议把体质降到8，原因包括：

- **反魔法力场 / Antimagic Field**可能压制护符；
- 护符可能被夺走；
- 可能临时解除同调；
- 自然体质仍决定护符失效后的生命值与体质豁免。

## 2.4 法师技能 / Wizard Skills

建议选择：

- **奥秘 / Arcana**；
- **洞悉 / Insight**。

人类额外技能已经提供：

- **调查 / Investigation**。

2级法师能力：

- **学者 / Scholar**  
  选择**调查 / Investigation**获得**专精 / Expertise**。

**命运恩惠 / Boon of Fate**兼顾：

- 战斗中的关键豁免；
- 敌人的关键豁免；
- 战后的神格碎片调查；
- 每次掷先攻或完成短休/长休后恢复。

它比只用于位移的**次元旅行恩惠 / Boon of Dimensional Travel**更适合本战役的长期成长机制。

# 5. 免费珍稀物品与同调  
# Free Rare Items and Attunement

## 5.1 免费珍稀一：健康护符 / Amulet of Health

- 稀有度：珍稀 / Rare
- 需要同调 / Requires Attunement
- 体质固定为19。
- 是本构筑压低自然体质、释放标准矩阵的核心。

## 5.2 免费珍稀二：传送头盔 / Helm of Teleportation

- 稀有度：珍稀 / Rare
- 需要同调 / Requires Attunement
- 3充能；
- 消耗1充能施放：
  - **传送术 / Teleport**；
- 每日黎明恢复1d3充能。

主要用途：

- 战斗结束后撤离；
- 返回固定基地；
- 避免强制保留7环法术位；
- 当队伍陷入无法继续战斗的状态时作为团体逃生。

## 5.3 额外购买：移位斗篷 / Cloak of Displacement

- 价格：4,000 gp
- 稀有度：珍稀 / Rare
- 需要同调
- 对你的攻击通常具有劣势；
- 受到伤害后暂时失效，直到你下回合开始；
- 速度为0时失效。

## 5.4 默认同调位

1. **健康护符 / Amulet of Health**
2. **传送头盔 / Helm of Teleportation**
3. **移位斗篷 / Cloak of Displacement**

# 7. 预言学派与免费法术规则  
# Diviner and Free Spell Rules

## 7.1 普通法师免费法术

- 1级法师获得6个1环法术；
- 此后每获得1个法师等级，向法术书加入2个法师法术；
- 每个法术必须是当时拥有法术位的环阶；
- 20级普通法师总计：
  - 6 + 19×2 = **44个法术**。

## 7.2 预言学者 / Divination Savant

3级获得：

- 两个不高于2环的预言学派法术。

之后每次首次获得新的法术位环阶时：

- 免费加入一个当前能够施放的预言学派法术。

按本团采用的解释，额外获得：

- 3级：两个1–2环预言法术；
- 获得3环位时：一个；
- 获得4环位时：一个；
- 获得5环位时：一个；
- 获得6环位时：一个；
- 获得7环位时：一个不高于7环的预言法术；
- 获得8环位时：一个；
- 获得9环位时：一个。

总计额外：

- **9个预言法术**。

最终最低法术书规模：

- 44 + 9 = **53个法术**。

# 14. 拟像术计划  
# Simulacrum Plan

## 14.1 正常施放优先于祈愿复制

若有12小时准备时间和1,500 gp红宝石粉末：

1. 准备：
   - **拟像术 / Simulacrum**
   - **预知术 / Foresight**
   - **心灵屏障 / Mind Blank**
   - **法术无效结界 / Globe of Invulnerability**
   - **力场墙 / Wall of Force**
   - **迷宫术 / Maze**
2. 用7环位正常施放**拟像术 / Simulacrum**；
3. 完成后，拟像复制当时的角色数据；
4. 本体随后完成长休，恢复自己的全部法术位；
5. 拟像不能通过休息恢复资源，但保留独立行动、反应与专注。

这种方式的优势：

- 本体进入下一场战斗时保留9环位；
- 拟像可以使用自己的9环位施放：
  - **预知术 / Foresight**；
- 本体保留：
  - **祈愿术 / Wish**。

## 14.2 拟像的战前职责

根据战斗类型选择：

- 给本体施放：
  - **预知术 / Foresight**，持续8小时；
- 或给本体施放：
  - **心灵屏障 / Mind Blank**，持续24小时；
- 战斗中维持：
  - **法术无效结界 / Globe of Invulnerability**
  - 或**力场墙 / Wall of Force**
  - 或其他专注法术。

本体与拟像分别维持一个专注法术。

## 14.3 需要确认的DM裁定

提前确认：

- 拟像是否复制当前剩余法术位；
- 已消耗的7环位是否也在拟像上视为已消耗；
- 拟像是否拥有另一组**预兆 / Portent**骰；
- 拟像是否可以使用史诗恩惠。

构筑不依赖“拟像额外获得三枚预兆骰”才能成立。

# 15. 祈愿术用法  
# Wish Uses

## 15.1 基础规则

**祈愿术 / Wish**可以安全复制：

- 任意职业；
- 任意8环或以下法术；
- 不需要满足通常施法要求；
- 不需要昂贵材料；
- 复制法术属于基础用途，不触发非常规祈愿反噬。

## 15.2 默认优先级

### 情况A：当前没有可用拟像

用**祈愿术 / Wish**复制：

- **拟像术 / Simulacrum**

优点：

- 立即生效；
- 不需要12小时；
- 不消耗1,500 gp红宝石粉末。

缺点：

- 消耗本体9环位；
- 拟像可能复制本体已经消耗9环位后的状态，取决于DM裁定。

### 情况B：已有可用拟像

根据战斗选择：

1. **英雄宴 / Heroes’ Feast**
   - 长时间团队增益；
   - 免除昂贵材料；
   - 适合大型首领战。

2. **神圣灵光 / Holy Aura**
   - 8环牧师法术；
   - 强力团队防御；
   - 需要专注。

3. **敌意/共感 / Antipathy/Sympathy**
   - 可用于固定基地或特定敌人；
   - 长时间效果。

4. **复生类法术 / Resurrection Spells**
   - 队伍发生严重死亡时紧急使用。

## 15.3 是否每次都用祈愿复制拟像？

不是。

若正常拟像仍存活且法术资源充足：

- 不应重复制造；
- 保留祈愿术应对本场战斗。

若拟像资源耗尽、严重受伤或已经死亡：

- 用正常施法重建最节省9环位；
- 没有12小时准备时间时，再用祈愿复制。

- **防护结界 / Guards and Wards**

# 17. 可靠返回基地  
# Reliable Return to Base

## 17.1 传送头盔＋关联物

**传送术 / Teleport**使用“关联物 / Linked Object”时可以准确抵达。

关联物定义：

- 在过去六个月内；
- 从目的地取走的物品。

基地应准备大量标注日期的小物品，例如：

- 接收室地砖碎片；
- 基地书房中的旧书；
- 墙壁金属片；
- 接收室专用徽章；
- 基地实验室中的容器。

操作：

1. 每名核心队员携带至少一件关联物；
2. 每五个月统一更换；
3. 使用**传送头盔 / Helm of Teleportation**或自己施放**传送术 / Teleport**；
4. 目的地固定为A区传送接收室。

## 17.2 为什么暂时不建设永久传送法阵？

永久**传送法阵 / Teleportation Circle**需要：

- 每日在同一地点施放；
- 连续365天；
- 每次消耗50 gp；
- 总成本18,250 gp。

当前使用关联物已经可以准确返回，因此不值得在初始40,000 gp中投入18,250 gp。

以后资金充足时可以建设，但法阵必须位于A区，而不是休息区。

## 17.3 紧急撤离顺序

1. **传送头盔 / Helm of Teleportation**
2. 自己准备的**传送术 / Teleport**
3. **江湖骗子斗篷 / Cape of the Mountebank**
4. **任意门 / Dimension Door**
5. 无法返回时使用：
   - **莫登肯宏伟宅邸 / Mordenkainen’s Magnificent Mansion**

- 战斗法术位全部恢复；
- AC增益持续；
- 约29点临时生命；
- 准备位被换回战斗法术。

2. **力场墙 / Wall of Force**

4. 根据弱项使用：
   - **放逐术 / Banishment**
   - **变形术 / Polymorph**
   - **怪物定身术 / Hold Monster**

低预兆骰用于：

- 强制敌人关键豁免失败。

高预兆骰用于：

- 保证自己的关键豁免；
- 保证队友重要攻击；
- 视DM裁定用于神格碎片调查。

## 20.2 敌方法师

1. 先手建立：
   - **法术无效结界 / Globe of Invulnerability**
2. 保留反应：
   - **法术反制 / Counterspell**
3. 使用低预兆骰影响敌方：
   - 2024版法术反制引发的体质豁免；
   - **心智迷乱 / Befuddlement**
4. 需要摧毁力场时：
   - **解离术 / Disintegrate**

## 20.3 大量敌人

优先：

- **催眠图纹 / Hypnotic Pattern**
- **冰雨暴 / Sleet Storm**
- **缓慢术 / Slow**
- **突触静止 / Synaptic Static**
- **重力反转 / Reverse Gravity**
- **流星爆 / Meteor Swarm**

## 20.4 近距离爆发

组合：

1. 高环施放：
   - **召唤次级元素 / Conjure Minor Elementals**
2. 靠近目标15尺；
3. 下一轮高环施放：
   - **灼热射线 / Scorching Ray**
4. 每次命中触发额外伤害；
5. 使用：
   - **迷踪步 / Misty Step**
   - **任意门 / Dimension Door**
   - 或其他位移撤离。

这是高风险爆发方案，不是默认控制路线。

# 21. 神格碎片调查  
# Divine Fragment Investigation

## 21.1 基础加值

人类主方案：

- 智力21：+5
- 调查专精：+12
- 总计：
  - **调查 / Investigation +17**

常规支持：

- 队友使用**协助 / Help**：优势；
- 牧师施放**指导术 / Guidance**：+1d4。

达到DC 30的概率约为：

- 优势 + 指导术 +17：
  - **77.1%**

若检定仍失败，再使用：

- **命运恩惠 / Boon of Fate**：+2d4

总体成功率约为：

- **94.0%**

还可以使用：

- **英雄激励 / Heroic Inspiration**
- **幸运儿 / Lucky**

进一步降低失败风险。

## 21.2 预兆骰 / Portent

需要提前与DM确认：

- 预兆骰替换调查检定，是否仍算“掷出D20获得碎片”；
- 是否必须由玩家实际掷骰。

若允许预兆替换：

- 高预兆骰可以保留给战后调查；
- 但在战斗中出现团灭风险时，应优先用于保命。

## 21.3 第一个属性成长目标

如果神格碎片允许属性+1：

> 第一目标：智力21 → 22

立即获得：

- 法术DC 19 → 20；
- 法术攻击+11 → +12；
- 调查+17 → +18。

# 22. 战后与长休流程  
# Post-Combat and Long-Rest Routine

1. 立即确认战场安全。
2. 由牧师施放：
   - **指导术 / Guidance**
3. 队友使用：
   - **协助 / Help**
4. 完成：
   - **智力（调查）/ Intelligence (Investigation)**
   - 搜索神格碎片。
5. 必要时使用：
   - **命运恩惠 / Boon of Fate**
   - **英雄激励 / Heroic Inspiration**
6. 使用：
   - **传送头盔 / Helm of Teleportation**
   - 加基地关联物返回A区。
7. 在A区完成身份确认与解除潜在追踪。
8. 进入B区缓冲走廊。
9. 进入C区休息区。
10. 仪式施放：
    - **警报术 / Alarm**
11. 施放：
    - **莫登肯忠犬 / Mordenkainen’s Faithful Hound**
12. 确认：
    - **警戒武器 / Weapon of Warning**
    - 由守夜队友或基地守卫同调，用于遇袭后提高全队先攻；
    - 真正的睡眠预警依赖**警报术 / Alarm**与**莫登肯忠犬 / Mordenkainen’s Faithful Hound**。
13. 完成长休。

# 23. 后续没有神格碎片时的兼职  
# Multiclass Progression without a Divine Fragment

> 20级以上的职业成长不是标准职业表的一部分，必须服从DM的史诗等级规则。
